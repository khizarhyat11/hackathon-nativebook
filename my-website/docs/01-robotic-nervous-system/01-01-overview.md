# Module 01 — Overview

**Estimated time:** 6–10 hours

## Learning Objectives

- Understand ROS 2 core abstractions: nodes, topics, services, actions, and TF.
- Implement simple `rclpy` nodes and timers.
- Author a minimal URDF biped model and visualize it in `rviz2`.
- Prepare a reproducible lab that demonstrates a publish/subscribe heartbeat and sensor topics.

## Lessons

- 1.02 — ROS 2 Architecture (fundamentals)
- 1.03 — rclpy & Agent Bridging (practical patterns)
- 1.04 — URDF Anatomy & XACRO (modeling)
- 1.05 — Lab: Hello Robot (hands-on)
- 1.06 — Assessment & Deliverables

:::tip
Prerequisites: Python 3.10+, basic Linux/WSL familiarity, and a ROS 2 distribution (Humble or later) installed. Use a container or VM if you prefer isolation.
:::
