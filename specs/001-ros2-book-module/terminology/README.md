# ROS 2 Module Terminology

This directory contains terminology definitions for the ROS 2 Book Module.

## Glossary

| Term | Definition |
|------|------------|
| Node | Executable that uses ROS 2 client libraries |
| Executor | Manages execution of callbacks |
| DDS | Data Distribution Service middleware |
| Topic | Named bus for publish/subscribe communication |
| Service | Request/response communication pattern |
| Action | Extended service with feedback and cancellation |
| QoS | Quality of Service policies |
| URDF | Unified Robot Description Format |
| rclpy | ROS 2 Python client library |
| Callback | Function called when event occurs |
| Publisher | Entity that sends messages to a topic |
| Subscriber | Entity that receives messages from a topic |
